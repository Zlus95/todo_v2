# todo_v2

A simple Tudu sheet application with authorization and registration

User can:
* Create the task
* Edit status of the task(to do/doing/done)
* Delete the task

## Core Technologies

### Backend

1) Golang
2) MongoDB
3) golang-jwt
4) gorilla/mux
5) swagger
6) Fresh

### Frontend

1) ReactJS
2) react-query
3) tailwindcss
4) axios
